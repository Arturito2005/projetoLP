{\em companies} & A pointer to the \doxylink{struct_companies}{Companies} struct. \\
\hline
{\em branchs} & A pointer to the \doxylink{struct_branch___activity}{Branch\+\_\+\+Activity} struct.\\
\hline
\end{DoxyParams}
This function hides or deletes comments from an existing company in the \doxylink{struct_companies}{Companies} struct. The function checks if there are companies before attempting to hide or delete comments. If the company has comments, the function allows the user to choose between hiding or deleting a comment. If the user chooses to delete a comment, the function asks for confirmation before proceeding with the deletion. If the user chooses to hide a comment, the function allows the user to choose between hiding or unhiding the comment. After a comment is hidden or deleted, the function saves the updated \doxylink{struct_companies}{Companies} struct to a file.


\begin{DoxyParams}{Parameters}
{\em companies} & A pointer to the \doxylink{struct_companies}{Companies} struct that stores all companies. \\
\hline
{\em branchs} & A pointer to the \doxylink{struct_branch___activity}{Branch\+\_\+\+Activity} struct that stores all activity branches.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The function will print a message and return if there are no companies to hide/delete comments. 
\end{DoxyNote}
