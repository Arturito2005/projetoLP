void report_classi ( Companies companies)

Generates a report on company classifications.

Parameters
companiesThe Companies struct.

Generates a report on company classifications.

This function calculates the average rating of each company and identifies the company(s) with the highest and lowest average ratings. It then prints a report detailing the average ratings of all companies and highlighting the company(s) with the highest and lowest averages.

Parameters
companiesstruct Companies that contains all the companies and their ratings.
Note
If there is only one company, the function will simply calculate and print its average rating.

◆ report_comment()

void report_comment ( Companies companies)

Generates a report on company comments.

Parameters
companiesThe Companies struct.

Generates a report on company comments.

This function calculates the total number of comments for each company and identifies the company(s) with the highest and lowest number of comments. It then creates a report detailing the total number of comments of all companies and highlighting the company(s) with the highest and lowest number of comments. At the beginning of the report, it shows the data of all companies with comments and all the comments of them.

Parameters
companiesStruct Companies, which contains the information of all companies and their respective comments.
Note
If there is only one company, the function will simply print its comments.

void report_search ( Search_comp search_comp)

Generates a report on company searches.

Parameters
search_compThe Search_comp struct.

Generates a report on company searches.

It calculates the total number of times the user searched for a company, and then shows how many times each search method was used.

Then it compares how many times each option was used and shows the most and least used option(s) or if both were used the same number of times.

Parameters
search_compStruct Search_comp, which contains the counter for each search method.
Note
If only one search method was used in total, that method will be presented.
