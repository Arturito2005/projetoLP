Compares the branch activity with the given branch.

This function iterates over all the branches in the Branch_Activity struct and compares the given branch name with the name of each branch. If a match is found, it prints a message indicating that the branch already exists and returns 1. If no match is found, it prints a success message and returns 0.

Parameters
branchsThe Branch_Activity struct.
branch_oThe name of the branch to be compared with existing branches.
Returns
int Returns 1 if the branch already exists, and 0 otherwise.
Note
The function uses the strcmp function to compare the branch names.

Searches for the position of the branch in the branch activity.

This function iterates over all the branches in the Branch_Activity struct and compares the given branch name with the name of each branch. If a match is found, it returns the position of the branch in the struct. If no match is found, it prints an error message and returns -1.

Checks if the branch is used in the given companies.

This function iterates over all the companies in the Companies struct and compares the given branch name with the branch name of each company. If a match is found, it prints a message indicating that the branch is being used by at least one company and returns 1. If no match is found, it returns 0.

name_branchThe name of the branch to be checked.
Returns
int Returns 1 if the branch is being used by any company, and 0 otherwise.
