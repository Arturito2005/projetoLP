# Projeto LP
## English
### 📌 Introduction
This repository contains my final project for the Programming Laboratory course. The project involves developing a business directory application that allows users to discover, rate, and interact with industrial and service companies located within a given geographic region.  
The project documentation is in HTML, generated by Doxygen.

---
### 📁 Folder Structure

- **Projeto**: this folder contains all the code for my Programming Languages (LP) project, organized into the following subfolders:  
  - **build**: stores intermediate files generated during the compilation process, such as object files (.o) and temporary dependencies.  
  - **dist**: contains the final files ready for distribution, such as executables, libraries, and resources used by the application.  
  - **doxygen**: folder with all the documentation generated by Doxygen.  
  - **nbproject**: contains the project configuration files used by NetBeans IDE, including build properties, project structure, and user settings.  
- **Enunciado_Projeto.pdf**: the practical assignment brief.

---
### 🛠️ Languages and Tools

#### 💻 Languages
- **C++**: A high-performance, general-purpose programming language used for developing the core logic of the application.

#### 🧰 Tools
- **NetBeans**: An integrated development environment (IDE) that supports C++ development with code editing, debugging, and build tools.
- **Doxygen**: A documentation generator that produces structured API documentation from annotated C++ source code.
