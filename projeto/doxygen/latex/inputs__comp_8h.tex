Fills the fields of a company in the \doxylink{struct_companies}{Companies} struct.

This function is used in the \textquotesingle{}add\+\_\+companies\textquotesingle{} and \textquotesingle{}edit\+\_\+companies\textquotesingle{} voids, where its main objective is to take advantage of the common fields of these options and reduce the size of the code. This function collects various pieces of information about a company, validates them, and stores them in the \doxylink{struct_companies}{Companies} struct. It prompts the user to enter the company\textquotesingle{}s name, category, branch of activity, address (street, locality and postal code), and state (active or inactive). It also validates the branch of activity and postal code using the \textquotesingle{}validation\+\_\+branch\textquotesingle{} and \textquotesingle{}validation\+\_\+cod\+\_\+postal\textquotesingle{} functions, respectively. If the entered branch of activity or postal code is not valid, it will continue to prompt the user to enter a valid one. After all the information is collected and validated, it calls the \textquotesingle{}information\+\_\+company\textquotesingle{} function to display the information of the newly created company.
