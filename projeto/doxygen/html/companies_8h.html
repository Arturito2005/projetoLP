◆ add_companies()

void add_companies ( Companies * companies,
Branch_Activity branchs )

Adds companies to the Companies struct.

Parameters
companiesA pointer to the Companies struct.
branchsThe Branch_Activity struct.

This function adds a new company to the Companies struct. The function checks if there are valid activity branches before adding the company. If the current capacity of the Companies struct is reached, the function reallocates memory to accommodate more companies. The function also initializes the company's comments and ratings.

Parameters
companiesA pointer to the Companies struct that stores all companies.
branchsThe Branch_Activity struct that stores all activity branches.
Note
The function will print an error message and return if memory reallocation fails.
The function will print a message and return if there are no active activity branches.

◆ consult_companies()

void consult_companies ( Companies companies)

Consults companies in the Companies struct.

Parameters
companiesThe Companies struct.

This function consults an existing company in the Companies struct. The function checks if there are companies before attempting to consult one. If the company to be consulted is found, the function prints the company's information, including comments and ratings. The function also calculates and prints the average of the company's ratings.

Parameters
companiesThe Companies struct that stores all companies.
Note
The function will print a message and return if there are no companies to consult.

◆ delete_companies()

void delete_companies ( Companies * companies)

Deletes companies from the Companies struct.

Parameters
companiesA pointer to the Companies struct.

This function deletes an existing company from the Companies struct. The function checks if there are companies before attempting to delete one. If the company to be deleted has no comments, the function asks the user for confirmation before proceeding with the deletion. If the company to be deleted has comments, the function prints a message and does not delete the company. After a company is deleted, the function saves the updated Companies struct to a file.

Parameters
companiesA pointer to the struct Companies that stores all companies.
Note
The function will print a message and return if there are no companies to delete.

◆ ocul_del_companies()

void ocul_del_companies ( Companies * companies,
Branch_Activity * branchs )

Hides or deletes comments from companies in the Companies struct.

Parameters
companiesA pointer to the Companies struct.
branchsA pointer to the Branch_Activity struct.

This function hides or deletes comments from an existing company in the Companies struct. The function checks if there are companies before attempting to hide or delete comments. If the company has comments, the function allows the user to choose between hiding or deleting a comment. If the user chooses to delete a comment, the function asks for confirmation before proceeding with the deletion. If the user chooses to hide a comment, the function allows the user to choose between hiding or unhiding the comment. After a comment is hidden or deleted, the function saves the updated Companies struct to a file.

Parameters
companiesA pointer to the Companies struct that stores all companies.
branchsA pointer to the Branch_Activity struct that stores all activity branches.
Note
The function will print a message and return if there are no companies to hide/delete comments.
