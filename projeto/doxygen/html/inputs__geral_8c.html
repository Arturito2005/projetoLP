◆ compare_nif()

int compare_nif ( Companies companies,
int nif_v )

Checks if the NIF (Fiscal Identification Number) provided by the user exists in the Companies struct.

Compares a NIF with the NIFs in the Companies struct.

It uses a loop to go through the entire Companies struct to check if the NIF of any of the companies stored in the struct is equal to the NIF entered by the user. If they are equal, it prints a message indicating that the entered NIF belongs to the company and returns the index 'i' of the company in the struct. If "nif_v" is not equal to the NIF of the current company and the loop has reached the last company in the struct, it prints an error message indicating that the NIF was not found and returns -1.

Parameters
companiesThe 'Companies' struct that contains all created companies.
nif_vThe NIF to be compared with the NIFs of the companies.
Returns
The index of the company if the NIF is found, -1 otherwise.

void readString ( char * string,
unsigned int size,
char * msg )

Reads a string and removes extra spaces.

Reads a string from the user.

This function displays a message to the user, reads a string from stdin and stores it in the string variable. If the read string ends with a newline character, it is replaced by a null character. If the read string does not end with a newline character, the cleanInputBuffer() function is called to clear the input buffer. If the read string is empty, a message is displayed to the user and the reading is repeated. Finally, the remove_extra_spaces() function is called to remove any extra spaces from the read string.

Parameters
stringThe string in which the user's input should be stored. The string is passed by reference, therefore, the changes made in the function will affect the original string.
sizeThe maximum number of characters to be read, including the final null character.
msgThe message to be displayed to the user before reading the input.

◆ repet_option()

int repet_option ( int op,
char * option,
char * structs )

This function performs the repetition of an option based on user input Then, it enters a do-while loop that continues until "rep" is 'S', 's', 'N' or 'n'. Inside the loop, it uses a switch case structure to handle different cases based on the value of "op". If "op" is 0, it prints a confirmation message. If "op" is 1, it prints a general message with "option" and "structs". Then, it assigns a character value to "rep" using the "getChar()" function. If "rep" is a lowercase letter, it converts it to uppercase. If "rep" is not 'S' or 'N', it prints an invalid option message. If "rep" is 'S', the function returns 0, and repeats the user's option. If "rep" is 'N', the function returns 1. The loop continues until "rep" is 'S' or 'N'.

Repeats an option.

Parameters
opThe operation code to determine the type of message to be printed.
optionA pointer to a character string representing the option.
structsA pointer to a character string representing the structs.
Returns
0 if the repetition character is 'S', 1 if it is 'N'.
